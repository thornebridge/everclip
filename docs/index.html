Built Different

Everything you copy,
organized forever.

EverClip intelligently captures, categorizes, and serves your clipboard history. No subscriptions. No cloud. No compromise.

🔍

Intelligent Detection

Automatically recognizes text, URLs, code, images, colors, Markdown, and files. Each type gets its own visual treatment and smart preview.

📌

Pinboards & Tags

Organize clips into color-coded pinboards. Tag entries across multiple collections. Pin items so they persist forever.

⚡

Smart Rules

Auto-route clipboard entries to pinboards based on source app, content type, regex patterns, or URL domain. Zero manual sorting.

📚

Paste Stack

Collect multiple copies with ⌘⇧C, then paste them all in order. Perfect for data entry, form filling, and batch workflows.

🔄

Paste Transformations

Paste as plain text, UPPERCASE, lowercase, Title Case, URL encoded, trimmed, or wrapped in quotes. Right-click any entry to transform.

🔎

Instant Search & Filter

Find anything instantly. Filter by type, source app, or search across all content. Keyboard-first navigation through your entire history.

⌨️

Text Expansion

Create snippets with abbreviation triggers. Use template variables like {{date}}, {{time}}, and {{clipboard}} for dynamic content.

🔒

Privacy First

Everything stays on your Mac. Exclude sensitive apps like password managers. Pause capture anytime. Your clipboard, your rules.

Built for the Mac. Only the Mac.

EverClip is a native Swift application compiled for Apple Silicon. No Electron. No web views. No compromise. Just pure, screaming-fast, native performance.

Not available on Windows, Linux, or iOS. But it's open source — fork it, port it, make it yours. We'd love to see what you build.
